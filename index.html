Dúvidas sobre Front-End

O que é um Desenvolvedor Front-End?

Um Desenvolvedor front-end é aquele que Desenvolve sites para web, para Client-side, ou seja, do lado do cliente!

Quais tecnologia um desenvolvedor Front-end precisa aprender?

Para se tornar um desenvolvedor front-end, precisa dominar algumas tecnologia como: HTML5, CSS3 e a linguagem de programação JavaScript.
